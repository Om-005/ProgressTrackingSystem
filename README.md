# 📌 Progress Tracking System

A full-stack **Progress Tracking System** that helps users manage, monitor, and analyze their tasks and productivity efficiently. This project is built using modern web technologies with a clean UI and scalable backend architecture.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 📊 Track progress of tasks/projects
- ✅ Create, update, and delete tasks
- 📈 Real-time progress monitoring
- 🎯 Dashboard for better visualization
- 🌐 Responsive UI for all devices

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/Om-005/ProgressTrackingSystem.git
cd ProgressTrackingSystem
```

---

### 2️⃣ Setup Backend

```bash
cd backend
npm install
```

Create a `.env` file in backend:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Run backend:

```bash
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
```
